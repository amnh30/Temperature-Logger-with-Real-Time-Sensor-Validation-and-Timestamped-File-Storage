To set the Program In Startup mode
Steps:

1-Press Win + R, type shell:startup, and press Enter. This will open the Startup folder.
2-Right-click in the folder, select New > Shortcut.
3-Browse to your .exe file, select it, and click Next.
4-Name the shortcut, and click Finish.
